# 📚 Advanced Library Management System (Java Swing)

## 🔹 Description
This is a desktop-based Library Management System developed using Java Swing.  
It allows users to manage books in a simple GUI interface with file storage.

## 🔹 Features
- Add new books
- Delete books
- Search books by ID, title, or author
- Issue and return books
- Show available/issued status
- Sorting by title and author
- Overdue tracking (based on issue date)
- Data saved using file handling (books.txt)

## 🔹 Technologies Used
- Java
- Swing (GUI)
- File Handling (Text File Storage)
- Object-Oriented Programming (OOP)

## 🔹 How to Run
1. Compile the program:
   javac AdvancedLibraryGUI.java

2. Run the program:
   java AdvancedLibraryGUI

## 🔹 Note
This project uses a single Java file and stores data locally in a text file (books.txt).
